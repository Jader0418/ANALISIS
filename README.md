# Algoritmos de Análisis y Diseño

Este repositorio contiene implementaciones de varios algoritmos en Python, organizados en diferentes categorías. Está dividido en algoritmos voraces (greedy) y algoritmos de análisis (sorting y búsqueda).

## Estructura del Proyecto

### Archivos en la raíz:
- `algoritmo_voraz.py`: Implementación del problema de la mochila fraccionaria usando algoritmo voraz (con errores intencionales para análisis).
- `conferencias.py`: Algoritmo voraz para selección de actividades (problema de conferencias).
- `moneda.py`: Algoritmo voraz para el problema del cambio de monedas.

### Carpeta `ANALISIS/`:
- `Busqueda_Binaria.py`: Implementación de búsqueda binaria.
- `Insertion_Sort.py`: Implementación del algoritmo de ordenamiento por inserción.
- `Quick_Sort.py`: Implementación del algoritmo Quick Sort con medición de tiempo.
- `README.md`: Análisis de modelo de costo en nube vs on-premise.

## Requisitos

- Python 3.x
- No se requieren bibliotecas externas adicionales.

## Cómo ejecutar

Cada archivo contiene un ejemplo de uso al final. Para ejecutar un algoritmo específico:

```bash
python algoritmo_voraz.py
python conferencias.py
python moneda.py
python ANALISIS/Quick_Sort.py
# etc.
```

## Descripción de los Algoritmos

### Algoritmos Voraces
- **Mochila Fraccionaria**: Selecciona objetos para maximizar el beneficio dado un peso máximo. (Nota: La implementación actual tiene errores para fines educativos).
- **Selección de Actividades**: Selecciona el máximo número de actividades no superpuestas.
- **Cambio de Monedas**: Encuentra la cantidad mínima de monedas para un monto dado.

### Algoritmos de Ordenamiento y Búsqueda
- **Quick Sort**: Algoritmo de ordenamiento rápido con complejidad O(n log n) promedio.
- **Insertion Sort**: Algoritmo de ordenamiento por inserción.
- **Búsqueda Binaria**: Búsqueda eficiente en listas ordenadas.
